# Niibot Frontend

React + TypeScript + Vite 前端。

## 啟動

```bash
cp .env.example .env
npm install
npm run dev
```

## 結構

```
src/
├── api/          # API 調用
├── components/   # 元件
├── pages/        # 頁面
├── contexts/     # React Context
└── lib/          # 工具函數
```

## 環境變數

```env
VITE_API_URL=http://localhost:8000
```
